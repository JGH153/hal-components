Getting started with HAL

This guide explains how to setup your Angular project to begin using HAL-Components.

Automatic Installation HAL

To add HAl simply open your terminal inside your project folder and copy the command from the terminal example, and run the command. It will add HDD & HAL, and ask if you want to add the required styling in your styles.scss

Manual Installation HAL

HAL is available as a NPM package here: npmjs.com/package/@hafslundnett/hal-components
Install with yarn or npm like:

  • Yarn: yarn add @hafslundnett/hal-components
  • Npm: npm install @hafslundnett/hal-components
