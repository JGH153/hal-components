Manual Installation HAL (Not recommended)

HAL is available as a NPM package here: npmjs.com/package/@hafslundnett/hal-components
Install with yarn or npm like:

  • Yarn: yarn add @hafslundnett/hal-components
  • Npm: npm install @hafslundnett/hal-components

Addiontal packages that are needed for Hal work are HDD-Style, Angular Material and CDK. You can install HDD-style through yarn or npm:

  • Yarn: yarn add @hafslundnett/hdd-style
  • Npm: npm install @hafslundnett/hdd-style

To install Angular Material use the install schematic. This will install both Angular Material and CDK. In your project root folder open your termninal and run the following command:
