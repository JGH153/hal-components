Getting started with HAL

This guide explains how to setup your Angular project to begin using HAL-Components.

Automatic Installation HAL

To add HAl simply open your terminal inside your project root folder and copy the command from the terminal example, and run the command. It will add HDD & HAL, and ask if you want to add the required styling in your styles.scss

Manual Installation HAL (Not recommended)

HAL is available as a NPM package here: npmjs.com/package/@hafslundnett/hal-components
Install with yarn or npm like:

  • Yarn: yarn add @hafslundnett/hal-components
  • Npm: npm install @hafslundnett/hal-components

You also need to have HDD-style installed, you can do this through yarn or npm:

  • Yarn: yarn add @hafslundnett/hdd-style
  • Npm: npm install @hafslundnett/hdd-style
