Getting started with HAL

This guide explains how to setup your Angular project to begin using HAL-Components. It includes information on installing HAL, and optionally displaying a sample HAL component in your application to verify your setup.

Installing HAL

WITH NG ADD:

To add HAl simply open your terminal inside your project folder and copy the command from the terminal example(ng add @hafslundnett/hal-components), and run the command.

It will add HDD & HAL, and ask if you want to add the required styling in your styles.scss

Add HAL to your angular project with ng add @hafslundnett/hal-components. It will add HDD & HAL, and ask if you want to add the required styling in your styles.scss

HAL is available as a NPM package here: npmjs.com/package/@hafslundnett/hal-components
Just install with yarn and start using it!

The github repo can be found here: github.com/hafslundnett/hal-components
Please submit any issues there directly.
