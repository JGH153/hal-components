HAL Components Documentation

Welcome to HAL, the component library for HDD-Style. This library consist of a series of ready to use components that will help you build websites faster that ever.

To add HAl simply open your terminal inside your project folder and copy the command:
ng add @hafslundnett/hal-components. It will add HDD & HAL to your project, and ask if you want to add the required styling in your styles.scss

HAL is available as a NPM package here: npmjs.com/package/@hafslundnett/hal-components
Just install with yarn and start using it!

The github repo can be found here: github.com/hafslundnett/hal-components
Please submit any issues there directly.
