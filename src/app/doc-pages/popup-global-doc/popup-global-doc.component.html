Popup global service

This is the documentation for PopupGlobalService. The service has methods for opening, closing and setting default size of an overlay. The overlay will appear in the middle of the screen. To close the overlay use the detach method.

✅ When to use:

The global popup should be used when you want to display information or actions as a popup that are not directly connected to a visual element on the page. The popup is more generic and made for use in non-specific scenarios, but can also be customized if needed.

⛔ When not to use:

Dialogs should not be used for information that is not relevant to the user's goals. 

import {{ '{' }} PopupGlobalService {{ '}' }} from '@hafslundnett/hal-components';
