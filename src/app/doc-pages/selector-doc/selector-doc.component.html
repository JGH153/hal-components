Selector / Dropdown component

Dropdowns are useful for organizing and showing the user a filtered selection.

✅ When to use:

A dropdown should be used if the user wants to make a selection from a set of predefined choices. The recommended selection for the user should be selected as default. If the list of choices is longer than 5, then an explanatory placeholder text should be used. The label should clearly describe the action, and the placeholder text should not consist of any additional information compared to the label. This as the placeholder is only visible before making a selection.

⛔ When not to use:

This dropdown should not be used when there are less than 3 alternatives or the numbers of alternatives are so large its becoming hard to find the right choice.
