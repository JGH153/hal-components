Paginator Component

Pagination enables tables content (client or server-side) to be divided up into smaller pages that can be navigated using this paginator.

When to use:

When a table with many rows needs to be divided into easier to handle pages

When not to use:

When the content is not a table. 

import {{ '{' }} PaginatorModule {{ '}' }} from '@hafslundnett/hal-components';
 
import {{ '{' }} Pagination {{ '}' }} from '@hafslundnett/hal-components';
