Autocomplete

The Autocomplete component can be attached to input elements in order to provide easy to use autocomplete functionality with keyboard navigation (arrow up/down, enter, escape). The options are filtered automatically based on what you provide to the component. The autocomplete part is an overlay, and do not push content below. The autocomplete are just suggestions and do not prevent the user from entering something not in the list of choices. Use a select if you want to limit the user to the choices. 

import {{ '{' }} AutocompleteDocComponentModule {{ '}' }} from '@hafslundnett/hal-components';
