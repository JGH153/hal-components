Popup connected component

Popups / Dialogs are a "conversation" between the user and the system, and are used where information or action is required from the user. They should be focused, direct and helpful.

PopupConnectedComponent is a popup which is connected to an origin element. The default position of the popup is above the origin it is connected to. To be able to connect to an origin the CdkOverlayOrigin directive should be used on the origin element. The popup is also animated to appear and dissapear smoothly.

This feature is build on top of the CDK Overlay API:
https://material.angular.io/cdk/overlay/api

✅ When to use:

The connected popup should be used when you want to display information or action(s) that's connected to an element.

⛔ When not to use:

The connected popup should not be used for information that is not relevant to the user's goals. 

import {{ '{' }} PopupConnectedModule {{ '}' }} from '@hafslundnett/hal-components';
 
import {{ '{' }} OverlayModule {{ '}' }} from '@angular/cdk/overlay';
