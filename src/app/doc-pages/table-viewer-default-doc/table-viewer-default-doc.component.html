Table viewer default

This is a table for displaying dynamic and highly configurable data. The goal for this table system is that everything should be defined in typescript and make it easier to customise on the fly. It consists of three core elements: The html element, a config file describing the table and the actuall data.

This is the default version, but you can extend the engine directly to create an even more custom table if you want to. This is not yest documented

For pagination, set the paginationOptions to the desired elements pr page.

NB! Add the table inside a container with horizontal scroll if you want a table that scrolls horizontally due to it large size 

import {{ '{' }} TableViewerDefaultModule {{ '}' }} from '@hafslundnett/hal-components';
 
import {{ '{' }} TableEngine {{ '}' }} from '@hafslundnett/hal-components';
