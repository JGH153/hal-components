Feedback service

Alert messages are messages with information that is important or time sensitive. The goal is to inform and redirect. Should be clear, communicative, descriptive, short, contextual and have the right color.

When to use:

Alert messages are used to guide the user. For an error, inform the user what the problem is. What went wrong? Why? For an alert, give them information about the status and how this can lead to an error. What can go wrong?

When not to use:

Alert messages should not be used unnecessarily. 

import {{ '{' }} FeedbackModule {{ '}' }} from '@hafslundnett/hal-components';
