Feedback service

This is the documentation for Feedback service. To use the service, call the "showMessage" method and include the arguments for message, severity, closeable and duration of the feedback message. The feedback messages appear by default in the top right corner of th UI.

Typical use: When displaying some kind of information after an action. The feedback messages are categorises in four main topics; success, information, warnings and error messages. 

import {{ '{' }} FeedbackModule {{ '}' }} from '@hafslundnett/hal-components';
 
import {{ '{' }} FeedbackService {{ '}' }} from '@hafslundnett/hal-components';

Inject and import the FeedbackService service in the component that utilizes the service
