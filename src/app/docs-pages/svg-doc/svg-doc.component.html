SVG component

This is the documentation for the hal-svg-element component. The component can be utilised whenever you need a visual element like an icon or picture in your application. The size and color can be defined directly in the component, letting you easily customize the svg element to your need. 

import {{ '{' }} SvgElementModule {{ '}' }} from '@hafslundnett/hal-components';
