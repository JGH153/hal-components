HAL Schematic

Angular schematics allows us to generate HAL specific components and elements for your projects. It also allows for auto installing and updating of HAL.

Add HAL to your angular project with ng add @hafslundnett/hal-components. It will add HDD & HAL, and ask if you want to add the required styling in your styles.scss

All commands ask for any information it needs like name and type. This is done so you don't have to remember the arguments
