Closeable-Row component

This is the documentation for the hal-closeable-row component. The component can be utilized in any use cases where you want a closeable or collapseble row. This can either hide or show information or functionality from the user with a click. 

import {{ '{' }} CloseableRowModule {{ '}' }} from '@hafslundnett/hal-components';

Examples

Closeable-Row Header | No inputs The content inside the body of the closeable-row.

Closeable-Row Header | No padding on the row body The content inside the body of the closeable-row.

Closeable-Row Header | Start as expanded set to false The content inside the body of the closeable-row.
