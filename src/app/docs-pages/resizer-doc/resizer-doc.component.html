Resizer component

This is the documentation for Resizer component. The component can be used in use cases where you want to enable the user to resize the element in UI. Attach the component through a html id, and optionally set the sizes for how much you want to resize the chosen elements.

Since the components is linked to an external html element, some style changes might be necessary to give the component a logical position. This is most evident for the vertical resizer that should be inline with the element you want to resize. Consider creating a container for the element and the resizer as in the example below. 

import {{ '{' }} ResizerModule {{ '}' }} from '@hafslundnett/hal-components';
