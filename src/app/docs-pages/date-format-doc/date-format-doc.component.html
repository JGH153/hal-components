Date Constants

The Date Constants are a selection of date formats to display date objects. We first present how to import the date constants, the available formats and at bottom section see how to utilize the DatePipe with the date constants. 

import {{ '{' }} DateConstants {{ '}' }} from '@hafslundnett/hal-components';

Available Formats

Date Constants Format Result
{{ row.constant }} {{ row.format }} {{ row.result }}

To format date objects with the appropriate date constant/format add the 'date-fns' package to your project and use the 'format' method. Click here for more info on the : date-fns package

In Html/Angular

Necessary imports: After importing the DateConstants, to enable to use date formats with a norwegian language, you need to import packages for unicode identifiers. In the root app.module of your project, import the following packages, and add the localID to the providers list: 

import {{ '{' }} registerLocaleData {{ '}' }} from '@angular/common';
 
import localeNb from '@angular/common/locales/nb';
 
providers: [ {{ '{' }} provide: LOCALE_ID, useValue: 'nb' {{ '}' }} ],;

Examples

Todays date in csvDateFormat: {{ today | date: dateConstants.csvDateFormat }}

Todays date in dotddmmyyhhmm: {{ today | date: dateConstants.dotddmmyyhhmm }}

Todays date in ddDashMM: {{ today | date: dateConstants.ddDashMM }}
