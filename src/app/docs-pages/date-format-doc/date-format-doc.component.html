Date Constants

The Date Constants are a selection of date formats to display date objects. Use Angular Date Pipe, and its method 'formatDate' to store the date to be displayed. In the 'formatDate' method, pass the DateConstant as the format. More information on the DatePipe can be found here: DatePipe

Necessary imports: To be able to use date formats with a norwegian language, you need to import packages for unicode identifiers. In the root app.module of your project, import the following packages: 

import {{ '{' }} registerLocaleData {{ '}' }} from '@angular/common';
 
import localeNb from '@angular/common/locales/nb';

Then in the root app.module providers, insert the object: {{ '{' }} provide: LOCALE_ID, useValue: 'nb' {{ '}' }}
When the necessary imports are implemented, import DateConstants as shown below in the component who needs the date format. 

import {{ '{' }} DateConstants {{ '}' }} from '@hafslundnett/hal-components';
