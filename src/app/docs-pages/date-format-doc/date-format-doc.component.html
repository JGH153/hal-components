Date Constants

The Date Constants are a selection of date formats to display date objects. Use Angular Date Pipe, and its method 'formatDate' to store the date to be displayed. In the 'formatDate' method, pass the DateConstant as the format. More information on the DatePipe can be found here: DatePipe 

import {{ '{' }} DateConstants {{ '}' }} from '@hafslundnett/hal-components';
