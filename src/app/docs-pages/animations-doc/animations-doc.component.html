Animations

This is the documentation for Animation constants. These are standard animation directives to be used directly on elements for initation, displaying list, or to create drop down effects.

To enable the use of the animations import them in the component where they are used. You could also import several animations in the same import line if you intend to use multiple animaitons. 

import {{ '{' }} scaleUp {{ '}' }} from '@hafslundnett/hal-components';
 
import {{ '{' }} scaleDown {{ '}' }} from '@hafslundnett/hal-components';
 
import {{ '{' }} listAnimation {{ '}' }} from '@hafslundnett/hal-components';
 
import {{ '{' }} curtainDown {{ '}' }} from '@hafslundnett/hal-components';
 
import {{ '{' }} heightDown {{ '}' }} from '@hafslundnett/hal-components';
