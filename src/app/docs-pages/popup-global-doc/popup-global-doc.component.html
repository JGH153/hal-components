Popup global service

This is the documentation for PopupGlobalService. The service has methods for opening, closing and setting default size of an overlay. The overlay will appear in the middle of the screen. To close the overlay use the detach method.

Typical use: Displaying some kind of information without any actions. The popup is more generic and made for use in non-specific scenarios, but can also be customized if needed. 

import {{ '{' }} PopupGlobalService {{ '}' }} from '@hafslundnett/hal-components';
