Popup global service

This is the documentation for PopupGlobalService. The service has methods for opening and setting default size of an overlay. The overlay will appear in the middle of the screen. To close the overlay use the detach method. 

import {{ '{' }} PopupGlobalService {{ '}' }} from '@hafslundnett/hal-components';

Code TS

123

Additional Info

Method Descirption
setupOverlay(maxWidth) Sets the max-width of the popup.
openOverlay(overlayRef, overlayComponent) Opens the overlay(popup) by passing the OverlayRef and the overlayComponent to display.
detach(overlayRef) Detaches the overlay by passing the OverlayRef.
