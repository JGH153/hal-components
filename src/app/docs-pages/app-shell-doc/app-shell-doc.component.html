App Shell

All applications need a shell that includes page name, navigation and the current user. The app shell provides all these features with minimal work. This helps all applications to have a shared design and user experence.

The app shell is designed to work with the router, but you can just provide a single page/component if you want to. Two interfaces is also provided for the current user, and side menu elements 

import {{ '{' }} AppShellModule {{ '}' }} from '@hafslundnett/hal-components';
 
import {{ '{' }} MenuElement, User {{ '}' }} from '@hafslundnett/hal-components';
