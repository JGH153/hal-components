Selector / Dropdown component

The select / dropdown component should be used if the user needs a list of predefined choices. The placeholder of the input field is by default the same as the label. If a different placeholder is wanted, it is possible with an input. No placeholder should consist of information the label is missing, as the placeholder is only visible before selecting. To be able to use the SelectData interface import it from hal-components. The interface is on the format value: string, viewValue: string. 

import {{ '{' }} Selector {{ '}' }} from '@hafslundnett/hal-components';
 
import {{ '{' }} SelectData {{ '}' }} from '@hafslundnett/hal-components';
