Loading spinner component

This is the documentation for the hal-loading-spinner component. The loading-spinner can be used in use cases where users have to wait until further interactions are available. 

import {{ '{' }} LoadingSpinnerModule {{ '}' }} from '@hafslundnett/hal-components';

Examples

The loading spinner comes defualt in color '--hdd-blue', and does not have any other inputs that to call the selector tag 'hal-loading-spinner' to create it.
