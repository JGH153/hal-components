Popup connected component

This is the documentation for PopupConnectedComponent. This is a popup which is connected to an origin element! The default position of the popup is above the origin it is connected to.

Typical use: Displaying some kind of information where the popup is connected to an element. 

import {{ '{' }} PopupGlobalService {{ '}' }} from '@hafslundnett/hal-components';
