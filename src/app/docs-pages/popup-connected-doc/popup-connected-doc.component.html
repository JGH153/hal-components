Popup connected component

This is the documentation for PopupConnectedComponent. This is a popup which is connected to an origin element! The default position of the popup is above the origin it is connected to. To be able to connect to an origin the CdkOverlayOrigin derective should be used on the origin element. The popup is also animated to appear and dissapear smoothly.

This feature is build on top of the CDK Overlay API:
https://material.angular.io/cdk/overlay/api

Typical use: Displaying some kind of information where you want the popup to be connected to an element. 

import {{ '{' }} PopupConnectedModule {{ '}' }} from '@hafslundnett/hal-components';
