Keyboard Shortcuts Service

The service lets you enable keyboard-shortcuts all over in the application. Enabling the service in your app-component makes the shortcuts available everywhere. You therefore only have to enable the service once!

To open the overview of shortcuts press ctrl and space. This overview lists all the shortcuts specifically for that component. In each component it is therefore possible to add new shortcuts with the method getKeyEventsForKey(), that only exists in that component. 

import {{ '{' }} KeyboardShortcutsService {{ '}' }} from '@hafslundnett/hal-components';
 
import {{ '{' }} KeyName {{ '}' }} from '@hafslundnett/hal-components';

Examples

Press ctrl-space to see the overview of shortcuts in this component. This shortcut works everywhere.

Press ctrl-a to see the shortcut added in this component, open console log to see result.
