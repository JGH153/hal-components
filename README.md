## Getting started
1. `yarn`
2. `yarn watch-hal`
3. (in new terminal): `yarn start`
4. Open http://localhost:4201/



## Publish to NPM

- Build with `yarn build-hal`
- Naviagte into dist/hal-components/
- run `npm publish`
